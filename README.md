# Lander-Electronics
Electronics for 25kg TVC hopper

### Creating new projects:
Open ```Template Project/Template Project.kicad_pro``` <br>
File>Save_As, then enter the new filename

### Adding New Symbols and Footprints
Adding footprints and symbols to the project specific library is the same as  usual. When connecting a footprint to a .stp file define a relative path to the file using ```${KIPRJMOD}/../Symbols and Footprints/3D Models/FILENAME.stp```
